# Glidely.
> A Premium, Enterprise-Grade Kanban Workspace.

Glidely is a high-performance project management application designed with a focus on visual clarity, real-time synchronization, and modern UI/UX aesthetics.

## ✨ Features
- **Enterprise Login:** Sleek split-pane authentication powered by Firebase Google Auth.
- **Glassmorphism UI:** Sophisticated Midnight Aurora theme with technical dot-grid patterns.
- **Recycle Bin System:** Intelligent task recovery logic with a 30-day retention window.
- **Real-time Collaboration:** Integrated team chat and online user presence indicators.
- **Responsive Design:** Mobile-first architecture with dynamic viewport scaling (dvh).

## 🛠 Tech Stack
- **Frontend:** XML, CSS (Custom Variables), JavaScript (ES6 Modules)
- **Database:** Firebase Firestore (v10)
- **Authentication:** Firebase Auth
- **Utilities:** SortableJS, Canvas-Confetti, Marked.js
